# i18n + RTL Rules

## Locale resolution

Order: `de-AT → de-DE → en`. Generic before regional only if regional is missing. Missing key fires `i18n_missing_key` (sampled 1%) and renders the `en` fallback — never the raw key.

## RTL mirroring (`dir="rtl"` when locale is `ar`, `he`, `fa`, `ur`)

| Element | LTR | RTL |
|---|---|---|
| Text column | left (cols 1–5) | right (cols 8–12) |
| Portrait | center-right (cols 5–9) | center-left (cols 4–8) |
| Profile card | right (cols 9–12) | left (cols 1–4) |
| Carousel dots | bottom-right | bottom-left |
| Stat icons | precede number | follow number |
| CTA arrow icons | `→` | `←` (mirror via `rtl:rotate-180`) |
| Logo strip scroll | left-to-right | right-to-left |
| Quote marks | `"…"` | `«…»` |

## Forbidden in RTL

- `text-left` / `text-right` hardcoded. Use `text-start` / `text-end`.
- `margin-left` / `padding-right`. Use `ms-*` / `pe-*` (Tailwind logical).
- Mirrored portraits. Portraits are art — do NOT flip.
- Mirrored numerals (`1,234` stays `1,234` even in RTL — locale-formatted via `Intl.NumberFormat`).

## Required languages for MVP

`en` (default), `de`, `ar`. Adding a 4th = `MINOR` bump.
